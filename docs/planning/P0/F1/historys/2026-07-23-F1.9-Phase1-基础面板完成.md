# F1.9 Phase 1 基础面板完成

> 最后更新：2026-07-23 | 版本：v1.0

## 变更摘要

F1.9 快捷粘贴面板 Phase 1（基础面板）TDD 实现完成。建立独立快速粘贴面板的基础设施，将全局快捷键触发行为从"唤起主窗口"改为"呼出独立 NSPanel 面板"。

## 实现内容

### 新增文件（8 个）

| 文件 | 职责 |
|------|------|
| `ClipMind/Models/QuickPasteSettings.swift` | 浮层超时配置持久化（默认 5 秒，范围 1-30 秒）+ 变更通知 |
| `ClipMind/UI/QuickPaste/QuickPastePanelController.swift` | NSPanel 控制器：创建、定位、显示、关闭、Esc/失焦关闭、位置记忆 |
| `ClipMind/UI/QuickPaste/QuickPasteView.swift` | SwiftUI 视图：搜索框 + LazyVStack 列表 + 默认高亮第一行 + 键盘事件路由 |
| `ClipMindTests/Models/QuickPasteSettingsTests.swift` | 配置默认值/范围/持久化/变更通知测试 |
| `ClipMindTests/App/GlobalHotkeyServiceQuickPasteTests.swift` | 快捷键触发行为变更测试 |
| `ClipMindTests/UI/QuickPastePanelControllerTests.swift` | 面板控制器定位/显示/关闭/位置记忆/Esc/失焦测试 |
| `ClipMindTests/UI/QuickPasteViewTests.swift` | 视图默认高亮/边界/空列表回车测试 |
| `ClipMindUITests/QuickPastePanelUITests.swift` | UI 测试：面板出现/定位/Esc/失焦（编译检查通过，CI 运行） |

### 修改文件（2 个）

| 文件 | 变更 |
|------|------|
| `ClipMind/App/GlobalHotkeyService.swift` | `handleHotkeyPressed()` 改发 `.openQuickPaste` 通知（原 `.openMainWindow`） |
| `ClipMind/App/ClipMindApp.swift` | AppDelegate 新增 `quickPastePanelController` 持有 + 监听通知 + 初始化控制器 + 设置 contentView |

### 计划缺口修复

- NSPanel 未设置 contentView（计划文件遗漏）：添加 `setContentView()` 方法，在 AppDelegate 中创建 QuickPasteView 并通过 NSHostingController 注入
