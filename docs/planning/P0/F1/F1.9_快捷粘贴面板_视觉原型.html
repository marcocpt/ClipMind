视觉要点： ① 双击 = 回车，触发相同粘贴流程； ② 有权限路径无浮层提示（用户感知"双击即粘贴"，无中间步骤）； ③ 面板先关闭再模拟粘贴按键（避免面板抢焦点拦截按键）； ④ 粘贴完成后无任何 UI 残留（面板消失、无浮层）。
场景 5：无权限时降级浮层提示"已复制，按 Cmd+V 粘贴" AC-F1.9-7
无辅助功能权限时，双击/回车触发降级流程：写入剪贴板 → 关闭快速粘贴面板 → 显示浮层提示。浮层在剪贴板被消费后消失，或超时兜底消失（默认 5 秒）。浮层不抢夺前台应用焦点。
终端（前台应用，未获得辅助功能权限）
$ git push origin feature/F1.9-quick-paste-panel

超时兜底：5 秒后消失（可配置）
视觉要点： ① 浮层文案固定为"已复制，按 ⌘V 粘贴"，不显示剪贴板原文（NFR-003 安全性）； ② 浮层定位在屏幕中央（或上次关闭位置）； ③ 浮层深色半透明背景 rgba(30,30,35,0.92) + 毛玻璃模糊，与系统通知风格一致； ④ 浮层不抢夺前台应用焦点（保持在前台应用之上但不激活）； ⑤ 消失条件：剪贴板被消费（变化计数变化）或超时兜底（默认 5 秒，可配置 1~30 秒）； ⑥ 两条消失路径互斥，先触发者生效。

场景 8：辅助功能权限被撤销时自动降级 AC-F1.9-12
用户曾获得辅助功能权限，在系统设置中撤销后，下次触发粘贴流程（双击或回车）自动走无权限降级路径，不尝试模拟粘贴按键。每次粘贴流程重新检测权限状态（不缓存）。

全局图例与说明

配色规范：

• 面板背景 #ffffff · 边框 #d1d1d1 · 阴影 0 8px 32px rgba(0,0,0,0.18)

• 高亮背景 #d4e8ff · 高亮左边框 #0a84ff（3px）

• 类型标签：文本蓝 #0a84ff · 图片橙 #fb923c · 文件路径绿 #34c759

• 提示条背景 #fff3cd · 提示左边框 #ffc107

• 降级浮层背景 rgba(30,30,35,0.92) + 毛玻璃模糊

• 文字主色 #1d1d1f · 次要 #6e6e70 · 弱化 #8a8a99


尺寸规范：

• 快速粘贴面板：360×480（与菜单栏 popover 视觉一致，固定尺寸不支持自定义）

• 降级浮层：自适应内容（约 280×52），屏幕中央定位

• 设置面板：max-width 560px（与现有设置面板视觉一致）

• 列表行内边距 12px · 行间距 0（边框分隔）· 类型标签字号 10px


交互键盘：

• ⌘⇧V 全局快捷键呼出面板（默认值，用户可配置）

• ↑↓ 方向键上下移动高亮

• ↩ 回车键触发粘贴（等同双击）

• Esc 关闭面板（不粘贴）

• 双击列表行触发粘贴流程

• ⌘V 降级路径中用户手动粘贴（消费浮层消失条件）


可访问性（NFR-005）：

• 所有交互元素设置无障碍标签（如"剪贴项列表"、"搜索框"、"已复制提示"）

• 高亮差异不仅依赖颜色（辅以左边框形状差异），色盲用户可识别

• 键盘导航完整覆盖所有操作（不依赖鼠标）


安全性（NFR-003）：

• 降级浮层不显示剪贴板原文，仅显示"已复制，按 ⌘V 粘贴"通用文案

• 日志不输出剪贴板原文，仅记录元数据（如"已写入剪贴板，长度 256"）

• 模拟粘贴按键仅发送系统标准粘贴按键，不发送任意按键序列
