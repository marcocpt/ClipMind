## 0.5 合规约束（AGENTS.md 第 10 节）

- 主 Scheme `ClipMind` 必须保持 App Store 合规（启用 App Sandbox + 最小必要 entitlement + 公开 API + 标准 TCC 流程）。
- 方案1（CGEvent 模拟 Cmd+V + AXUIElement 获取 caret）使用公开 API + 标准 TCC 权限流程，可在主 Scheme 实现，但需在设计文档风险章节标注：若 App Store 审核拒绝，则方案1暂存到 `ClipMind-Dev` Scheme。
- 方案3（无权限降级：写剪贴板 + 浮层提示 + 监听消费后消失）纯沙盒内实现，完全合规。
- 设计文档需包含「合规待定」标注章节（针对方案1的 Accessibility 权限依赖）。
