本测试用例表覆盖设计规范 v1.2 中定义的全部 7 条验收标准（F1.8-AC1 ~ F1.8-AC7），按功能模块组织：

| 兼容性与迁移 | AC7 | F1.8-AC7 |
| **合计** | **7** | — |

### 1.2 覆盖状态说明

| 状态 | 含义 | 图示 |
|------|------|------|
| COVERED | 已覆盖，测试代码已实现并通过 | ✅ |
| PARTIAL | 部分覆盖，测试代码部分实现 | 🟡 |

### 1.3 测试框架

| 框架 | 用途 | 适用范围 |
|------|------|---------|
| XCTest | 单元测试 | SampleDataSeeder、EncryptedStore 扩展、ClipItem Codable、语义搜索、迁移 |
| XCUITest | UI 自动化测试 | 首启注入后主窗口显示、清除示例数据按钮、类型标签 |
| 手动 | 人工验证 | 真实首启体验、截图/录屏证据 |

```

ClipMindUITests/
└── SampleDataUITests.swift                  # 首启显示示例、清除按钮、真实数据保留
```
