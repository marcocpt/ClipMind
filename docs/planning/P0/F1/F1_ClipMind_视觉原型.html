请求 权限

ClipMind 需要以下权限以提供完整功能。所有权限均可拒绝，拒绝后核心功能（分类、搜索、本地处理）仍可使用。

辅助功能权限（Accessibility）
用于获取前台 App bundleId，识别复制内容的来源 App。仅在复制时读取，不记录键盘输入。
已授权
通知权限（Notifications）
用于敏感内容忽略提示。仅在使用敏感识别功能时发送，不会推送广告或营销消息。
待授权
⚠ 说明：权限缺失不影响核心功能，但来源 App 会记录为 "unknown"，敏感忽略通知将不显示。

配置 API Key（可选）

配置云端 LLM API Key 以启用智能总结、即时翻译、智能改写、提取待办四项一键处理能力。分类与搜索基于本地模型，无需 API Key 即可使用，可跳过此步骤稍后配置。

隐私 默认安全

为了让 ClipMind 在首启即安全，以下隐私保护项已全部默认开启。你可以在设置中随时调整。

已为你开启的隐私保护
  • 敏感内容识别 · 自动识别密码 / Token / 验证码 / 银行卡 / 身份证，识别后不入库
  • 应用黑名单 · 1Password / 钥匙串 / 银行 / 支付宝 等敏感 App 自动忽略
  • 自动清理 · 30 天前的剪贴历史自动删除，防止长期累积敏感数据
  • 本地加密存储 · AES-256-GCM 加密，数据默认不出本机
  • 云端能力可选 · 仅在你主动点击"一键处理"时调用云端 LLM，分类与搜索全本地
